
1. Types de documents (définitions simples)
A) Facture (client particulier ou professionnel)
- Document de vente indiquant : vendeur, acheteur, date, numéro de commande, articles, montants, TVA (si applicable).

B) Facture commerciale (commercial invoice)
- Document utilisé pour formalités douanières / export, avec description marchandise, valeur, origine, etc. (selon pays).

C) Bon cadeau / reçu sans prix (gift receipt)
- Document destiné à offrir : peut exclure les prix (selon option disponible) et afficher un message cadeau.

2. Ce que la boutique peut fournir (règles par défaut)
2.1 Facture (standard)
- OUI, nous pouvons fournir une facture au format numérique (PDF) pour la plupart des commandes.
- Par défaut : facture envoyée sur demande, après expédition (ou après livraison selon cas).

2.2 Facture commerciale (export / douanes)
- OUI, si la commande est expédiée vers un pays nécessitant des documents douaniers et si les informations requises sont fournies.
- La facture commerciale peut être obligatoire pour certains transporteurs/destinations.

2.3 Bon cadeau / reçu sans prix
- OUI, sur demande AVANT expédition.
- Si la commande est déjà en statut “étiquette créée” ou “expédiée”, l’ajout d’un bon cadeau papier n’est généralement plus possible.
- Alternative : bon cadeau numérique (message) envoyé par e-mail (si demandé).

3. Quand demander (timing)
- Facture : idéalement dès la commande, ou au plus tard dans les 30 jours suivant la livraison.
- Facture commerciale : à demander immédiatement après commande (avant expédition), surtout pour l’international.
- Bon cadeau : à demander le plus tôt possible, impérativement avant expédition.

4. Comment demander (procédure)
Envoyez au service client :
- Numéro de commande
- Type de document demandé : “FACTURE” / “FACTURE COMMERCIALE” / “BON CADEAU”
- Mode souhaité : PDF par e-mail (recommandé) / document dans le colis (si possible)

4.1 Informations nécessaires pour une facture (particulier)
- Nom et prénom
- Adresse de facturation complète

4.2 Informations nécessaires pour une facture (professionnel)
- Raison sociale
- Adresse de facturation complète
- Numéro de TVA intracommunautaire (si applicable)
- Numéro SIRET (si applicable, France)
Remarque : sans informations complètes, la facture peut être émise au nom du destinataire de la commande.

4.3 Informations nécessaires pour une facture commerciale (export/douanes)
- Nom / société
- Adresse complète (facturation et/ou livraison)
- Numéro de téléphone
- E-mail
- Description de la marchandise (si besoin spécifique)
- Motif : “vente” / “cadeau” (selon cas)
Remarque : certains champs (origine, code HS, valeur) peuvent être standardisés selon nos données produit.

4.4 Informations pour bon cadeau
- Message cadeau (optionnel)
- Nom du destinataire (optionnel)
- Indication : “SANS PRIX” (si vous ne souhaitez pas afficher les prix)

5. Délais de fourniture (par défaut)
- Facture PDF : 1 à 2 jours ouvrés après validation des informations (souvent après expédition).
- Facture commerciale : avant expédition si possible ; sinon, envoyée en PDF dès émission.
- Bon cadeau papier : seulement si demandé avant expédition et si la logistique le permet.
- Bon cadeau numérique : 1 jour ouvré après demande (si informations complètes).

6. Règles & limites importantes
- Modifications : une facture peut être corrigée si erreur de données (nom/adresse/TVA) signalée rapidement.
- Confidentialité : nous pouvons masquer certains éléments non requis selon le document, mais les mentions légales obligatoires restent.
- Commandes multi-colis : une seule facture peut couvrir l’ensemble ; sur demande, nous pouvons préciser les expéditions.
- Plateforme : si la plateforme émet déjà un justificatif, notre facture boutique sert de document complémentaire (selon cas).

7. FAQ (réponses rapides)
Q : Puis-je avoir une facture ?
R : Oui. Envoyez-nous votre numéro de commande et vos informations de facturation, nous vous enverrons la facture en PDF.

Q : Je suis une entreprise, pouvez-vous mettre mon nom de société et TVA ?
R : Oui, envoyez raison sociale, adresse, TVA intracommunautaire (si applicable) et SIRET (si applicable) avec le numéro de commande.

Q : J’envoie la commande en cadeau, pouvez-vous retirer les prix ?
R : Oui, demandez un “bon cadeau / reçu sans prix” avant expédition. Si la commande est déjà expédiée, nous pouvons fournir un bon cadeau numérique (sans prix) par e-mail si disponible.

Q : J’ai besoin d’une facture commerciale pour la douane.
R : Envoyez le numéro de commande et les informations complètes (nom/société, adresse, téléphone). Nous émettrons une facture commerciale (PDF) si votre destination l’exige.
